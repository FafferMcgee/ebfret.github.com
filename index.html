ebFRET

ebFRET performs combined analysis on multiple single-molecule FRET time series to learn a set of rates and states

View My GitHub Profile

Consensus analysis of many single-molecule FRET time series

ebFRET is a library written in Matlab for the analysis of single-molecule FRET time series with hidden Markov models. It the successor of vbFRET, which it improves upon by allowing analysis of multiple time series at once. This approach, known as empirical Bayes, is both more accurate and statistically robust, and is suitable to more advanced experimental use cases, such as the detection of kinetically distinct sub-populations in an experiment.

Read the Paper

Hierarchically-coupled hidden Markov models for learning kinetic rates from single-molecule data Jan-Willem van de Meent (@jwvdm), Jonathan Bronson, Frank Wood, Ruben Gonzalez, Chris Wiggins (@chrishwiggins)

Example Usage

data = load_fret('my_data.dat')
eb = eb_fret(data.fret, [2,3,4], 10);
tracePlotter(data.fret, 'vit', eb.vit);
