ebFRET

ebFRET performs combined analysis on multiple single-molecule FRET time series to learn a set of rates and states

View My GitHub Profile

Consensus analysis of many single-molecule FRET time series

ebFRET is a library written in Matlab for the analysis of single-molecule FRET time series with hidden Markov models.

It is the successor of vbFRET, which it improves upon by performing combined analysis of multiple time series at once. This approach, known as empirical Bayes, is both more accurate and statistically robust. It also enables more advanced analysis use cases such as sub-population detection.

Example Usage

data = load_fret('my_data.dat');
eb = eb_fret(data.fret, [2,3,4], 10);
tracePlotter(data.fret, 'vit', eb(3).vit);
